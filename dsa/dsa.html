Array DSA

Problem Statement: Two Sum

Given an array of integers nums and an integer target, return the indices of the two numbers such that they add up to target.

You may assume that each input would have exactly one solution, and you may not use the same element twice. You can return the answer in any order.

Example:

Input: nums = [2, 7, 11, 15], target = 9
Output: [0, 1]
Explanation: nums[0] + nums[1] = 2 + 7 = 9

JavaScript Solution


function twoSum(nums, target) {
  const numToIndex = new Map();

  for (let i = 0; i < nums.length; i++) {
    const complement = target - nums[i];
    if (numToIndex.has(complement)) {
      return [numToIndex.get(complement), i];
    }
    numToIndex.set(nums[i], i);
  }
}

Explanation

  • Use a Map to store each number and its index while iterating through the array.
  • For each number, calculate its complement as target - current number.
  • If the complement exists in the map, return the indices of the complement and current number.
  • If not, add the current number and its index to the map and continue.
  • This approach runs in O(n) time, where n is the array length.
