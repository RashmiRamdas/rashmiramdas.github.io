Problem Statement: Two Sum

Given an array of integers nums and an integer target, return the indices of the two numbers such that they add up to target.

You may assume that each input would have exactly one solution, and you may not use the same element twice. You can return the answer in any order.

Example:

Input: nums = [2, 7, 11, 15], target = 9
Output: [0, 1]
Explanation: nums[0] + nums[1] = 2 + 7 = 9

JavaScript Solution


function twoSum(nums, target) {
  const numToIndex = new Map();

  for (let i = 0; i < nums.length; i++) {
    const complement = target - nums[i];
    if (numToIndex.has(complement)) {
      return [numToIndex.get(complement), i];
    }
    numToIndex.set(nums[i], i);
  }
}

Explanation

  • Use a Map to store each number and its index while iterating through the array.
  • For each number, calculate its complement as target - current number.
  • If the complement exists in the map, return the indices of the complement and current number.
  • If not, add the current number and its index to the map and continue.
  • This approach runs in O(n) time, where n is the array length.

Problem Statement: Best Time to Buy and Sell Stock

You are given an array prices where prices[i] is the price of a given stock on the ith day. You want to maximize your profit by choosing a single day to buy one stock and choosing a different day in the future to sell that stock. Return the maximum profit you can achieve from this transaction. If you cannot achieve any profit, return 0.

JavaScript Solution


function maxProfit(prices) {
  let minPrice = Infinity;
  let maxProfit = 0;

  for (let price of prices) {
    if (price < minPrice) {
      minPrice = price; // Step 1: Update minPrice if current price is lower
    } else if (price - minPrice > maxProfit) {
      maxProfit = price - minPrice; // Step 2: Update maxProfit if current profit is higher
    }
  }

  return maxProfit; // Step 3: Return the maximum profit found
}

Step-by-Step Explanation

  1. Initialize variables: Set minPrice to a very large number (Infinity) and maxProfit to 0.
  2. Iterate through each price: For each day's price in the array:
    • If the current price is less than minPrice, update minPrice to this lower price (meaning a better day to buy).
    • Otherwise, calculate the profit by subtracting minPrice from the current price. If this profit is greater than the current maxProfit, update maxProfit.
  3. Return the result: After checking all prices, return the maxProfit found. If no profit is possible, it will be 0.

Problem Statement: Contains Duplicate

Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.

JavaScript Solution


      var containsDuplicate = function(nums) {
        let set = new Set();
        for (let num of nums) {
          if (set.has(num)) {
            return true;
          }
          set.add(num);
        }
        return false;
      };

Explanation

  • Create a Set to keep track of unique numbers encountered.
  • Iterate through each number in the array:
    • If the number already exists in the set, return true immediately (duplicate found).
    • If not, add the number to the set.
  • If the loop completes without finding duplicates, return false.
  • This approach has a time complexity of O(n) and space complexity of O(n).
